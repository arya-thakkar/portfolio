Expense Tracker

This project uses HTML for structuring the webpage (inputs, buttons, and tables), CSS for styling and layout (colors, fonts, dark theme, and responsiveness), and JavaScript for interactivity and logic. JavaScript plays the main role by handling user input, converting salary into different currencies, adding or deleting expense rows, and calculating both total expenses and remaining income. It interacts with HTML elements using the DOM (getElementById, querySelector, and event listeners), performs calculations, validates inputs, and updates results dynamically without reloading the page. Overall, HTML builds the skeleton, CSS designs it, and JavaScript makes it functional.

2D Canvas Editor

This project is a lightweight, web-based canvas editor built with React, Fabric.js, and Firebase Firestore. It features a simple home page with a button to create new canvases, which generates unique URLs for each canvas session. The editor provides a comprehensive set of tools including shape creation (rectangles, circles), text addition, and a pen tool for freehand drawing. Users can manipulate objects by moving, resizing, rotating, and deleting them, with real-time editing capabilities. The application integrates with Firebase Firestore for persistence, allowing users to save their work and continue editing later by accessing the same URL. The project demonstrates clean component architecture, efficient state management, and thoughtful user experience design.

React Fabric.js Firebase Canvas API

NotInstagram - Social Media Website

This project is a frontend-only social media website built with React, designed to replicate the core functionality and user experience of Instagram with a custom UI. The application features multiple responsive pages including a landing page with smooth animations, login/signup pages with regex validation, a home feed displaying posts from mock JSON data, detailed post views with comments, user profiles with grid layouts, and a "Discover People" page. The project demonstrates professional React component architecture with separate folders for pages, reusable components, mock data, and assets. It focuses on clean design, responsive layouts, smooth CSS transitions, and a consistent color theme across all devices.
